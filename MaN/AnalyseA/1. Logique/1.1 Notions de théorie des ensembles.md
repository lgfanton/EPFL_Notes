
Un **ensemble** est une collection bien définie d'objets/éléments distincts.

On définit souvent un ensemble $E$ en listant ses éléments :

**Exemples 1.1.**
- $E = \{\leftarrow, \rightarrow, \uparrow\}$. L'ordre des éléments n'importe pas.
- $\mathbb{N} = \{0, 1, 2, 3, \ldots\}$, $\mathbb{N}^* = \{1, 2, 3, \ldots\}$, $\mathbb{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots\}$
- $\mathbb{Q} = \left\{ \frac{p}{q} \mid p \in \mathbb{Z},\ q \in \mathbb{N}^* \right\}$

Un ensemble peut aussi être défini via une propriété :

**Exemple 1.2.** L'ensemble des entiers positifs pairs est $E = \{2, 4, 6, 8, \ldots\}$, ou encore
$$E = \{ x \in \mathbb{N} \mid \exists k \in \mathbb{N},\ x = 2k \}$$

## Appartenance et inclusion

**Définition 1.4.**
- On écrit $x \in E$ pour indiquer que $x$ est un élément de $E$, et $x \notin E$ pour le contraire.
- L'**ensemble vide** $\emptyset$ est l'ensemble ne contenant aucun élément. $E = \emptyset \iff \forall x,\ x \notin E$.
- **Inclusion** : $A \subseteq E$ si et seulement si $\forall x \in A,\ x \in E$.

**Exemple 1.5.** Si $E = \{a, b, c\}$, alors $\{a, b\} \subseteq E$ mais $\{a, d\} \not\subseteq E$.

**Remarque 1.6.** Il faut distinguer $\in$ (appartenance d'un élément) et $\subseteq$ (inclusion d'un ensemble).

### 1.1.1 Union, intersection

**Définition 1.9.**
$$A \cap B = \{ x \in E \mid x \in A \text{ et } x \in B \}$$
$$A \cup B = \{ x \in E \mid x \in A \text{ ou } x \in B \}$$

**Exemple 1.10.** Si $A = \{-1, 0, 1, 2\}$ et $B = \{0, 1, 3\}$, alors :
$$A \cap B = \{0, 1\}, \qquad A \cup B = \{-1, 0, 1, 2, 3\}$$

**Exemple 1.11.** Si $A = ]-\infty, 1]$, $B = ]0, 2[$, $C = ]1, 3]$, $D = [3, 4]$, alors :

| Opération | Résultat |
|-----------|----------|
| $A \cap B$ | $]0, 1]$ |
| $A \cap C$ | $\emptyset$ |
| $A \cup C$ | $]-\infty, 3]$ |
| $B \cap C$ | $]1, 2[$ |
| $B \cup C$ | $]0, 3]$ |
| $C \cap D$ | $\{3\}$ |

On peut aussi définir des unions/intersections indexées :
$$\bigcup_{k \in I} A_k, \qquad \bigcap_{k \in I} A_k$$

**Exemple 1.13.** Si $A_k = [k, k+1]$, alors $\bigcup_{k \in \mathbb{N}} A_k = \mathbb{R}^+$ et $\bigcap_{k \in \mathbb{N}} A_k = \emptyset$.

### 1.1.2 Complémentaire

**Définition 1.14.** Soit $A \subseteq E$. Le **complémentaire** de $A$ dans $E$ est :
$$C_E(A) = \{ x \in E \mid x \notin A \}$$

Parfois noté $\bar{A}$ ou $A^c$. On a $C_E(\emptyset) = E$ et $C_E(E) = \emptyset$.

Propriétés (lois de De Morgan) :
$$\overline{A \cup B} = \bar{A} \cap \bar{B}, \qquad \overline{A \cap B} = \bar{A} \cup \bar{B}$$
