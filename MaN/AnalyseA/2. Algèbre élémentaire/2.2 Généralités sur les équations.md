# 2.2 Généralités sur les équations

Une **équation** en $x$ est une proposition de la forme $f(x) = g(x)$, où l'on cherche les valeurs de $x$ (l'**ensemble solution** $S$) qui rendent l'égalité vraie.

L'ensemble de définition est $D = D_f \cap D_g$.

### 2.2.1 Équations avec paramètres

Une équation peut dépendre d'un ou plusieurs **paramètres** (ex. $m \in \mathbb{R}$). On cherche alors $S$ en fonction de $m$.

Dans la pratique :
1. Identifier les cas selon les valeurs du paramètre.
2. Résoudre dans chaque cas.
3. Vérifier les solutions.
