# 2.3 Équations et inéquations linéaires

### 2.3.1 Cas général

Une équation linéaire en $x$ est de la forme $ax + b = 0$.

$$ax + b = 0 \Rightarrow \begin{cases} x = -\frac{b}{a} & \text{si } a \neq 0 \\ \text{pas de solution} & \text{si } a = 0,\ b \neq 0 \\ \text{tout } \mathbb{R} & \text{si } a = 0,\ b = 0 \end{cases}$$

Pour une **inéquation** $ax + b \leq 0$ : même démarche, mais attention au changement de sens si $a < 0$.

### 2.3.2 Avec paramètre

**Exemple.** Résoudre $mx = 1$ selon $m \in \mathbb{R}$.

- Si $m \neq 0$ : $x = \frac{1}{m}$, donc $S = \left\{ \frac{1}{m} \right\}$.
- Si $m = 0$ : $0 = 1$, impossible, $S = \emptyset$.

**Exemple (inéquation).** Résoudre $mx \leq 1$ selon $m$.

- Si $m > 0$ : $x \leq \frac{1}{m}$, donc $S = \left]-\infty, \frac{1}{m}\right]$.
- Si $m = 0$ : $0 \leq 1$, toujours vrai, $S = \mathbb{R}$.
- Si $m < 0$ : $x \geq \frac{1}{m}$, donc $S = \left[\frac{1}{m}, +\infty\right[$.
