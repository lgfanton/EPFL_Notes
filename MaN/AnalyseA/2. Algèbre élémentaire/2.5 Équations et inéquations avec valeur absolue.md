# 2.5 Équations et inéquations avec valeur absolue

Rappel : $|x| = \max(x, -x)$, et $|x| \geq 0$ pour tout $x$.

### 2.5.1 L'équation $|x| = a$

$$|x| = a \Leftrightarrow \begin{cases} x = a \text{ ou } x = -a & \text{si } a \geq 0 \\ \text{pas de solution} & \text{si } a < 0 \end{cases}$$

### 2.5.2 L'inéquation $|x| \leq a$

$$|x| \leq a \Leftrightarrow \begin{cases} x \in [-a, a] & \text{si } a \geq 0 \\ \emptyset & \text{si } a < 0 \end{cases}$$

### 2.5.3 L'inéquation $|x| \geq a$

$$|x| \geq a \Leftrightarrow \begin{cases} x \in ]-\infty, -a] \cup [a, +\infty[ & \text{si } a \geq 0 \\ \mathbb{R} & \text{si } a < 0 \end{cases}$$

### 2.5.4 Équations à valeurs absolues

**Méthode générale :** Étudier les cas selon le signe de l'expression à l'intérieur de $|\cdot|$.

**Exemple.** Résoudre $|2x - 1| = |x + 3|$.

- Cas $2x - 1 = x + 3$ : $x = 4$.
- Cas $2x - 1 = -(x+3)$ : $3x = -2$, $x = -\frac{2}{3}$.

$S = \left\{-\frac{2}{3}, 4\right\}$.

### 2.5.5 Inéquations $f(x) \leq g(x)$ et $f(x) \geq g(x)$

Résoudre séparément sur les domaines où chaque membre garde un signe constant.
