# 3.3 Les 4 équations élémentaires

### 3.3.1 L'équation $\cos(x) = a$

Condition : $a \in [-1, 1]$.

Il existe $\omega \in [0, \pi]$ tel que $\cos(\omega) = a$ (valeur principale). Alors :
$$\cos(x) = a \Leftrightarrow x = \pm\omega + 2k\pi, \quad k \in \mathbb{Z}$$

### 3.3.2 L'équation $\sin(x) = a$

Condition : $a \in [-1, 1]$.

Il existe $\omega \in \left[-\frac{\pi}{2}, \frac{\pi}{2}\right]$ tel que $\sin(\omega) = a$. Alors :
$$\sin(x) = a \Leftrightarrow x = \omega + 2k\pi \quad \text{ou} \quad x = \pi - \omega + 2k\pi, \quad k \in \mathbb{Z}$$

### 3.3.3 L'équation $\tan(x) = a$

Pour tout $a \in \mathbb{R}$ :

Il existe $\omega \in \left]-\frac{\pi}{2}, \frac{\pi}{2}\right[$ tel que $\tan(\omega) = a$. Alors :
$$\tan(x) = a \Leftrightarrow x = \omega + k\pi, \quad k \in \mathbb{Z}$$

### 3.3.4 L'équation $\cot(x) = a$

Pour tout $a \in \mathbb{R}$ :

Il existe $\omega \in ]0, \pi[$ tel que $\cot(\omega) = a$. Alors :
$$\cot(x) = a \Leftrightarrow x = \omega + k\pi, \quad k \in \mathbb{Z}$$
