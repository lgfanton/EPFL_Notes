# 3.4 Formules trigonométriques

### 3.4.1 Préparation : une rotation dans le plan

Une rotation d'angle $\omega$ dans $\mathbb{R}^2$ envoie le point $(x, y)$ sur :
$$\begin{pmatrix} x\cos\omega - y\sin\omega \\ x\sin\omega + y\cos\omega \end{pmatrix}$$

### 3.4.2 Formules d'addition

$$\begin{aligned}
\cos(\alpha + \beta) &= \cos\alpha\cos\beta - \sin\alpha\sin\beta \\
\cos(\alpha - \beta) &= \cos\alpha\cos\beta + \sin\alpha\sin\beta \\
\sin(\alpha + \beta) &= \sin\alpha\cos\beta + \cos\alpha\sin\beta \\
\sin(\alpha - \beta) &= \sin\alpha\cos\beta - \cos\alpha\sin\beta
\end{aligned}$$

**Formules de duplication :**
$$\cos(2\alpha) = \cos^2\alpha - \sin^2\alpha = 2\cos^2\alpha - 1 = 1 - 2\sin^2\alpha$$
$$\sin(2\alpha) = 2\sin\alpha\cos\alpha$$

**Formule de bissection :**
$$\cos^2\!\left(\frac{\alpha}{2}\right) = \frac{1 + \cos\alpha}{2}, \qquad \sin^2\!\left(\frac{\alpha}{2}\right) = \frac{1 - \cos\alpha}{2}$$

**Formules de Simpson (produit → somme) :**
$$\cos\alpha\cos\beta = \frac{1}{2}[\cos(\alpha-\beta) + \cos(\alpha+\beta)]$$
$$\sin\alpha\sin\beta = \frac{1}{2}[\cos(\alpha-\beta) - \cos(\alpha+\beta)]$$
$$\sin\alpha\cos\beta = \frac{1}{2}[\sin(\alpha+\beta) + \sin(\alpha-\beta)]$$

### 3.4.3 Dérivées des fonctions trigonométriques

$$(\cos\omega)' = -\sin\omega, \qquad (\sin\omega)' = \cos\omega, \qquad (\tan\omega)' = \frac{1}{\cos^2\omega}$$

### 3.4.5 Factorisation

$$a\cos(x) + b\sin(x) = \sqrt{a^2+b^2}\cdot\sin\!\left(x + \varphi\right)$$

où $\tan\varphi = \frac{a}{b}$ (ou $\cos\varphi = \frac{b}{\sqrt{a^2+b^2}}$).

### 3.4.6 Équations et inéquations trigonométriques linéaires

Pour résoudre $a\cos(x) + b\sin(x) = c$ :

1. Diviser par $\sqrt{a^2 + b^2}$.
2. Identifier $\sin(\omega) = \frac{a}{\sqrt{a^2+b^2}}$ et $\cos(\omega) = \frac{b}{\sqrt{a^2+b^2}}$.
3. L'équation devient $\sin(x + \omega) = \frac{c}{\sqrt{a^2+b^2}}$.
4. Résoudre si $\left|\frac{c}{\sqrt{a^2+b^2}}\right| \leq 1$.
