# 4.1 Forme cartésienne

### 4.1.1 Définition

On introduit le nombre **imaginaire** $i$ tel que :
$$i^2 = -1$$

Un **nombre complexe** est un objet de la forme $z = a + ib$ avec $a, b \in \mathbb{R}$.

**Définition 4.4.** Soit $z = a + ib \in \mathbb{C}$ :
- La **partie réelle** : $\text{Re}(z) = a$
- La **partie imaginaire** : $\text{Im}(z) = b$
- L'écriture $z = a + ib$ est la **forme cartésienne** (ou algébrique)

**Remarque 4.1.** Comme $i^2 = -1$, on a aussi $(-i)^2 = -1$. Donc $i$ et $-i$ sont les deux racines de $x^2 + 1 = 0$.

### 4.1.2 Identification entre $\mathbb{C}$ et $\mathbb{R}^2$

Tout $z = a + ib$ est identifié au point $(a, b) \in \mathbb{R}^2$ dans le **plan de Gauss** (plan complexe).

**Quelques lieux géométriques en écriture complexe :**
- Droite verticale $x = a$ : $\text{Re}(z) = a$
- Droite horizontale $y = b$ : $\text{Im}(z) = b$
- Cercle de centre $z_0$, rayon $r$ : $|z - z_0| = r$

### 4.1.3 Opérations sur les nombres complexes

Soient $z = a + ib$, $z' = a' + ib'$.

**1. Égalité :** $z = z' \Leftrightarrow \text{Re}(z) = \text{Re}(z')$ et $\text{Im}(z) = \text{Im}(z')$

**2. Addition :**
$$z + z' = (a + a') + i(b + b')$$

**3. Multiplication :**
$$zz' = (aa' - bb') + i(ab' + ba')$$

**4. Conjugué** $\bar{z} = a - ib$ :

**Propriété 4.7.**
1. $\overline{\bar{z}} = z$
2. $\overline{z + z'} = \bar{z} + \bar{z'}$
3. $\overline{zz'} = \bar{z}\bar{z'}$
4. $z\bar{z} = \text{Re}^2(z) + \text{Im}^2(z) = |z|^2$
5. $\bar{z} = z \Leftrightarrow \text{Im}(z) = 0 \Leftrightarrow z \in \mathbb{R}$

**5. Module :**
$$|z| = \sqrt{z\bar{z}} = \sqrt{a^2 + b^2}$$

Représente la distance de $z$ à l'origine. Propriétés :
$$|zz'| = |z||z'|, \qquad \left|\frac{z}{z'}\right| = \frac{|z|}{|z'|}, \qquad |z + z'| \leq |z| + |z'|$$

**6. Inverse :** Pour $z \neq 0$ :
$$\frac{1}{z} = \frac{\bar{z}}{|z|^2}$$

### 4.1.4 Puissances et racines

**Puissances de $i$ :**
$$i^0 = 1,\quad i^1 = i,\quad i^2 = -1,\quad i^3 = -i,\quad i^4 = 1, \ldots$$

**Racine carrée** de $z = a + ib$ : chercher $w = x + iy$ tel que $w^2 = z$. On résout le système :
$$\begin{cases} x^2 - y^2 = a \\ 2xy = b \\ x^2 + y^2 = |z| \end{cases}$$
