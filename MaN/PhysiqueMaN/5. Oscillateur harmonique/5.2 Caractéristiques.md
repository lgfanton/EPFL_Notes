# 5.2 Caractéristiques

| Grandeur | Formule | Unité |
|----------|---------|-------|
| Pulsation propre | $\omega_0 = \sqrt{k/m}$ | rad·s⁻¹ |
| Période | $T_0 = \frac{2\pi}{\omega_0}$ | s |
| Fréquence | $f_0 = \frac{1}{T_0} = \frac{\omega_0}{2\pi}$ | Hz |
| Amplitude | $C = \sqrt{A^2 + B^2}$ | m |
| Phase initiale | $\varphi = \arctan(-B/A)$ | rad |
