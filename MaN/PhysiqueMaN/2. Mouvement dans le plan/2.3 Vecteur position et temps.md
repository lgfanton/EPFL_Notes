# 2.3 Vecteur position et temps

**Vecteur position** : vecteur donnant la position de l'objet dans le référentiel.

$$\vec{r} = x\vec{e}_x + y\vec{e}_y = \begin{pmatrix}x\\y\end{pmatrix} = \begin{pmatrix}r\cos\alpha\\r\sin\alpha\end{pmatrix}, \qquad r = \|\vec{r}\| = \sqrt{x^2+y^2}$$

Symbole : $\vec{r}$ ; unité : m.

**Temps** : symbole $t$ ; unité : s. Le vecteur position est une fonction du temps : $\vec{r} = \vec{r}(t)$.

**Déplacement** : $\Delta\vec{r} = \vec{r}_2 - \vec{r}_1$ ; unité : m.

**Trajectoire** :
$$\Gamma = \left\{P \mid \exists\, t,\ \overrightarrow{OP} = \vec{r}(t)\right\}$$

**Rencontre de deux objets** : $\exists\, t_r,\ \vec{r}_{\text{objet1}}(t_r) = \vec{r}_{\text{objet2}}(t_r)$.
