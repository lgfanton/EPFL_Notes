# 2.5 Accélération

**Accélération instantanée** :
$$\vec{a}(t) = \lim_{\Delta t \to 0} \frac{\vec{v}(t+\Delta t) - \vec{v}(t)}{\Delta t} = \frac{d\vec{v}}{dt} = \dot{\vec{v}}(t), \qquad \text{unité : m·s}^{-2}$$

L'accélération est :
- le taux de variation de la vitesse par rapport au temps ;
- toujours dirigée vers **l'intérieur du virage** ;
- composante tangentielle : changement de norme de $\vec{v}$ ;
- composante normale : changement de direction de $\vec{v}$.

## Mouvement uniformément accéléré (MUA)

Pour $\vec{a}(t) = \vec{a}_0 = \text{cste}$ :
$$\vec{v}(t) = \vec{a}_0(t - t_0) + \vec{v}_0$$
$$\vec{r}(t) = \frac{1}{2}\vec{a}_0(t-t_0)^2 + \vec{v}_0(t-t_0) + \vec{r}_0$$

La trajectoire est une **parabole** d'axe parallèle à $\vec{a}_0$.

## Mouvement rectiligne uniforme (MRU)

Cas particulier avec $\vec{a}(t) = \vec{0}$ :
$$\vec{v}(t) = \vec{v}_0, \qquad \vec{r}(t) = \vec{v}_0(t-t_0) + \vec{r}_0$$
