# 4.7 Problème du billard

**Problème.** Une bille part d'un point $O = (-4, 5)$ dans la direction $(3, -9)$. Elle rebondit sur la droite $x + 2y = 0$, puis sur la droite $y = x$. Trouver les points de rebond.

**Méthode générale.** Pour trouver le point de rebond d'une trajectoire sur une droite $d$ :
1. Paramétrer la trajectoire : $P(t) = P_0 + t\cdot\vec{d}$
2. Trouver $t$ tel que $P(t) \in d$
3. La nouvelle direction est l'image de $\vec{d}$ par la réflexion d'axe $d$

**Étape 1 : Premier rebond sur $x + 2y = 0$.**

Trajectoire : $(x,y) = (-4,5) + t(3,-9)$. Condition $x + 2y = 0$ :
$$(-4+3t) + 2(5-9t) = 0 \Rightarrow -4+3t+10-18t = 0 \Rightarrow 6 - 15t = 0 \Rightarrow t = \frac{2}{5}$$
$$P_1 = (-4 + \tfrac{6}{5},\, 5 - \tfrac{18}{5}) = (-\tfrac{14}{5},\, \tfrac{7}{5}) = \tfrac{7}{5}(-2, 1)$$

**Étape 2 : Nouvelle direction après réflexion sur $x + 2y = 0$.**

La réflexion d'axe $x + 2y = 0$ a matrice $S = \frac{1}{5}\begin{pmatrix}3&-4\\-4&-3\end{pmatrix}$ (voir §6).
$$\vec{d}_2 = S\begin{pmatrix}3\\-9\end{pmatrix} = \frac{1}{5}\begin{pmatrix}3\cdot3 + (-4)(-9)\\(-4)\cdot3 + (-3)(-9)\end{pmatrix} = \frac{1}{5}\begin{pmatrix}45\\15\end{pmatrix} = \begin{pmatrix}9\\3\end{pmatrix}$$

**Étape 3 : Second rebond sur $y = x$.**

Trajectoire depuis $P_1 = (-14/5, 7/5)$ dans direction $(9,3)$ : $(x,y) = (-14/5 + 9s,\, 7/5 + 3s)$. Condition $y = x$ :
$$\frac{7}{5} + 3s = -\frac{14}{5} + 9s \Rightarrow \frac{21}{5} = 6s \Rightarrow s = \frac{7}{10}$$
$$P_2 = (-\tfrac{14}{5} + \tfrac{63}{10},\, \tfrac{7}{5} + \tfrac{21}{10}) = (\tfrac{35}{10},\, \tfrac{35}{10}) = (\tfrac{7}{2}, \tfrac{7}{2})$$
