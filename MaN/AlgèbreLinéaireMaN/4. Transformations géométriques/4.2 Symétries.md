# 4.2 Symétries

**Définition.** On dit que $f : \mathbb{R}^m \to \mathbb{R}^m$ est une **symétrie** si :
$$f \circ f = \text{id}_{\mathbb{R}^m} \qquad (\iff A^2 = I_m)$$

**Lien avec les projections.** Si $g$ est la projection sur $\text{Im}\, f$ parallèlement à $\ker f$, alors $f = 2g - \text{id}$.

Réciproquement, si $f$ est une symétrie, alors $g = \frac{1}{2}(f + \text{id})$ est une projection.

**Propriétés.** Une symétrie a les valeurs propres $+1$ (vecteurs fixes) et $-1$ (vecteurs renversés).

**Exemple.** Symétrie par rapport à l'axe $y = x$ dans $\mathbb{R}^2$ : $(x,y) \mapsto (y,x)$.
$$A = \begin{pmatrix}0&1\\1&0\end{pmatrix}, \quad A^2 = I_2 \checkmark$$
