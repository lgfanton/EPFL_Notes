# 2.2 Bases et coordonnées

## Base

Une **base** $B = \{v_1, \ldots, v_n\}$ de $\mathbb{R}^n$ est un ensemble de $n$ vecteurs librement indépendants (linéairement indépendants) et générateurs.

Tout $v \in \mathbb{R}^n$ s'écrit de façon **unique** $v = t_1 v_1 + \cdots + t_n v_n$. Les $t_i$ sont les **coordonnées** de $v$ dans $B$ :
$$[v]_B = \begin{pmatrix}t_1\\\vdots\\t_n\end{pmatrix}$$

**Exemple (base canonique de $\mathbb{R}^2$).** $v_1 = (1,0)$, $v_2 = (0,1)$. Alors $v = (x,y) = xv_1 + yv_2$, donc $[v]_{B.c.} = \begin{pmatrix}x\\y\end{pmatrix}$.

**Exemple (autre base de $\mathbb{R}^2$).** $v_1 = (0,2)$, $v_2 = (\frac{1}{3}, 0)$ — ces vecteurs forment une base car ils sont linéairement indépendants.

$v = (x,y) = x\cdot1\cdot(1,0) + y\cdot\frac{1}{2}\cdot(0,2) = 3x\cdot(\frac{1}{3},0) + \frac{y}{2}\cdot(0,2)$

donc $[v]_B = \begin{pmatrix}y/2\\3x\end{pmatrix}$.

> **⚠** $v \neq (y/2,\ 3x)$ ! Les coordonnées dans $B$ ne sont pas les composantes dans $\mathbb{R}^2$.
