# 2.3 Changement de bases

Soient $B = \{v_1, v_2, v_3\}$ et $B' = \{v_1', v_2', v_3'\}$ deux bases de $\mathbb{R}^3$.

La **matrice de changement de base** $P$ est définie par :
$$(v_1'\ v_2'\ v_3') = (v_1\ v_2\ v_3)P$$

i.e. $P = \left([v_1']_B \mid [v_2']_B \mid [v_3']_B\right)$.

**Changement de coordonnées :** $Q = P^{-1}$, et :
$$[v]_{B'} = Q\,[v]_B = P^{-1}[v]_B$$

**Exemple (base canonique).** Si $B$ est la base canonique $B.c.$ et $v_1 = (3,0,0)$, $v_2 = (0,0,\frac{1}{2})$, $v_3 = (0,-1,0)$, alors :
$$P = \begin{pmatrix}3&0&0\\0&0&-1\\0&\frac{1}{2}&0\end{pmatrix}$$

et $[v]_B = P^{-1}[v]_{B.c.}$ donne les coordonnées dans $B$.
