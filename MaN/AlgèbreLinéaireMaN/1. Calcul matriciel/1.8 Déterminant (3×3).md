# 1.8 Déterminant (3×3)

**Définition.** Soit $A \in M_3(\mathbb{R})$. On pose :
$$\det(A) = \begin{vmatrix} a_{1,1}&a_{1,2}&a_{1,3}\\a_{2,1}&a_{2,2}&a_{2,3}\\a_{3,1}&a_{3,2}&a_{3,3}\end{vmatrix}$$

$$= a_{1,1}a_{2,2}a_{3,3} + a_{1,2}a_{2,3}a_{3,1} + a_{1,3}a_{2,1}a_{3,2}$$
$$- a_{1,2}a_{2,1}a_{3,3} - a_{1,1}a_{2,3}a_{3,2} - a_{1,3}a_{2,2}a_{3,1}$$

## Astuces de calcul

**Astuce 1 (règle de Sarrus).** Répéter les deux premières colonnes à droite et sommer les diagonales :

$$\det(A) = \underbrace{a_{11}a_{22}a_{33}}_{\searrow} + \underbrace{a_{12}a_{23}a_{31}}_{\searrow} + \underbrace{a_{13}a_{21}a_{32}}_{\searrow} - \underbrace{a_{12}a_{21}a_{33}}_{\swarrow} - \underbrace{a_{11}a_{23}a_{32}}_{\swarrow} - \underbrace{a_{13}a_{22}a_{31}}_{\swarrow}$$

**Astuce 2 (développement par cofacteurs).** Développer selon la première colonne :
$$\det(A) = a_{1,1} \begin{vmatrix}a_{2,2}&a_{2,3}\\a_{3,2}&a_{3,3}\end{vmatrix} - a_{2,1}\begin{vmatrix}a_{1,2}&a_{1,3}\\a_{3,2}&a_{3,3}\end{vmatrix} + a_{3,1}\begin{vmatrix}a_{1,2}&a_{1,3}\\a_{2,2}&a_{2,3}\end{vmatrix}$$

## Propriété fondamentale

**Proposition.** $\det(A) = 0 \iff$ au moins une ligne (colonne) est combinaison linéaire des deux autres.
