# 1.6 Systèmes d'équations linéaires

Un système d'équations linéaires s'écrit matriciellement $A\begin{pmatrix}x\\y\end{pmatrix} = \begin{pmatrix}a\\b\end{pmatrix}$.

## Méthode de résolution par décomposition colonne-ligne

Si $\text{rg}(A) = 1$, on peut écrire $A = \vec{u} \cdot \vec{v}^t$ (produit d'une colonne par une ligne).

**Exemple.** $A = \begin{pmatrix}2&4\\3&6\end{pmatrix} = \begin{pmatrix}2\\3\end{pmatrix}(1\ 2)$

Alors $A\begin{pmatrix}x\\y\end{pmatrix} = \begin{pmatrix}a\\b\end{pmatrix}$ devient $\begin{pmatrix}2\\3\end{pmatrix}(x + 2y) = \begin{pmatrix}a\\b\end{pmatrix}$.

- **Condition d'existence :** $\begin{pmatrix}2\\3\end{pmatrix} \sim \begin{pmatrix}a\\b\end{pmatrix}$, i.e. $3a = 2b$.
- **Solution :** si $3a = 2b$, une solution facile est $\begin{pmatrix}a/2\\0\end{pmatrix}$.
- **Solution générale :** $S = \left\{\begin{pmatrix}a/2\\0\end{pmatrix} + t\begin{pmatrix}2\\-1\end{pmatrix} \mid t \in \mathbb{R}\right\}$ si $3a = 2b$, $S = \emptyset$ sinon.

Géométriquement, $S$ est une droite passant par $\begin{pmatrix}a/2\\0\end{pmatrix}$ de direction $\begin{pmatrix}2\\-1\end{pmatrix}$.

## Système $3 \times 3$

Pour $A$ de taille $3 \times 3$ et $\text{rg}(A) = 2$ :
$$A\begin{pmatrix}x\\y\\z\end{pmatrix} = \begin{pmatrix}a\\b\\c\end{pmatrix}$$

On échelonne la matrice augmentée $[A | \vec{b}]$ pour trouver les solutions.

**Interprétation géométrique.** Chaque équation définit un plan dans $\mathbb{R}^3$. La solution est l'intersection de ces plans :
- 3 plans avec un point commun → solution unique
- 3 plans avec une droite commune → infinité de solutions (droite)
- Pas de point commun → $S = \emptyset$
