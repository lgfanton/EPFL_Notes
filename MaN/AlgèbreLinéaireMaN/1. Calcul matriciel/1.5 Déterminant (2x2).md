# 1.5 Déterminant (2×2)

**Définition.** On appelle **déterminant** d'une matrice $A = \begin{pmatrix}\alpha_{1,1}&\alpha_{1,2}\\\alpha_{2,1}&\alpha_{2,2}\end{pmatrix} \in M_2(\mathbb{R})$ le nombre réel :
$$\det A = \begin{vmatrix}\alpha_{1,1}&\alpha_{1,2}\\\alpha_{2,1}&\alpha_{2,2}\end{vmatrix} = \alpha_{1,1}\alpha_{2,2} - \alpha_{1,2}\alpha_{2,1}$$

**Proposition.** On a :
$$\det A = 0 \iff \text{rg}\, A \le 1$$

*Preuve.* Si $A = \begin{pmatrix}0&0\\0&0\end{pmatrix}$ l'équivalence est claire. Sinon, supposons par exemple $\alpha_{1,2} \neq 0$. Alors $\text{rg}\, A \le 1 \iff \frac{\alpha_{2,2}}{\alpha_{1,2}}\alpha_{1,1} = \alpha_{2,1} \iff \alpha_{1,1}\alpha_{2,2} - \alpha_{1,2}\alpha_{2,1} = 0 \iff \det A = 0$. $\square$

**Proposition.** Pour $A, B \in M_2(\mathbb{R})$ :
$$\det(AB) = \det A \cdot \det B$$

*Preuve.* [Série 2, Exercice 9] $\square$
