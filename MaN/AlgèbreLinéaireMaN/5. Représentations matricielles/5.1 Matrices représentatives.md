# 5.1 Matrices représentatives

Soit $f : \mathbb{R}^m \to \mathbb{R}^p$ linéaire de matrice $A \in M_{p,m}(\mathbb{R})$ en bases canoniques, et soient :
- $\mathcal{B} = v_1, \ldots, v_m$ une base de $\mathbb{R}^m$, de matrice de changement de base $P$ (de $\mathcal{B}_{can}$ à $\mathcal{B}$)
- $\mathcal{B}' = v_1', \ldots, v_p'$ une base de $\mathbb{R}^p$, de matrice de changement de base $Q$ (de $\mathcal{B}_{can}$ à $\mathcal{B}'$)

**Définition.** La **matrice représentative** de $f$ dans les bases $\mathcal{B}$ et $\mathcal{B}'$ est :
$$\boxed{[f]_{\mathcal{B},\mathcal{B}'} = Q^{-1}AP}$$

Elle vérifie $[f(v)]_{\mathcal{B}'} = [f]_{\mathcal{B},\mathcal{B}'}\, [v]_{\mathcal{B}}$.

Si $\mathcal{B} = \mathcal{B}'$ (et $m = p$), on note simplement $[f]_\mathcal{B} = P^{-1}AP$.

**Interprétation.** Les colonnes de $[f]_{\mathcal{B},\mathcal{B}'}$ sont les coordonnées dans $\mathcal{B}'$ des images des vecteurs de $\mathcal{B}$ :
$$[f]_{\mathcal{B},\mathcal{B}'} = \Big([f(v_1)]_{\mathcal{B}'} \;\Big|\; \cdots \;\Big|\; [f(v_m)]_{\mathcal{B}'}\Big)$$

**Exemple.** $f : \mathbb{R}^2 \to \mathbb{R}^2$, $A = \begin{pmatrix}2&-1\\-6&3\end{pmatrix}$. Bases $\mathcal{B} = (3,4),(2,1)$ et $\mathcal{B}' = (1,-1),(2,1)$.
$$P = \begin{pmatrix}3&2\\4&1\end{pmatrix}, \quad Q = \begin{pmatrix}1&2\\-1&1\end{pmatrix}, \quad Q^{-1} = \frac{1}{3}\begin{pmatrix}1&-2\\1&1\end{pmatrix}$$
$$[f]_{\mathcal{B},\mathcal{B}'} = Q^{-1}AP = \frac{1}{3}\begin{pmatrix}1&-2\\1&1\end{pmatrix}\begin{pmatrix}2&-1\\-6&3\end{pmatrix}\begin{pmatrix}3&2\\4&1\end{pmatrix} = \begin{pmatrix}14&21\\4&6\end{pmatrix}$$
