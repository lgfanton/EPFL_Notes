# 2.3 Changement de base ($\mathbb{R}^2$)

Soient $\mathcal{B} = v_1, v_2$ et $\mathcal{B}' = v_1', v_2'$ deux bases de $\mathbb{R}^2$.

La **matrice de changement de base** de $\mathcal{B}_{can}$ à $\mathcal{B}$ est :
$$P = \Big([v_1]_{\mathcal{B}_{can}} \;\Big|\; [v_2]_{\mathcal{B}_{can}}\Big)$$

i.e. les colonnes de $P$ sont les coordonnées canoniques des vecteurs de $\mathcal{B}$ :
$$(v_1'\ v_2') = (v_1\ v_2)P \qquad \Rightarrow \qquad Q = P^{-1}$$

**Changement de coordonnées** de $\mathcal{B}$ à $\mathcal{B}'$ :
$$[v]_{\mathcal{B}'} = Q\, [v]_{\mathcal{B}}$$

**Exemple.** $v_1 = (1,0)$, $v_2 = (1,1)$ ($\mathcal{B}$), et $\mathcal{B}_{can}$.

$$P = \begin{pmatrix}1&1\\0&1\end{pmatrix}, \qquad Q = P^{-1} = \begin{pmatrix}1&-1\\0&1\end{pmatrix}$$

Pour $v = (x,y)$ : $[v]_{\mathcal{B}} = Q\begin{pmatrix}x\\y\end{pmatrix} = \begin{pmatrix}x-y\\y\end{pmatrix}$.
