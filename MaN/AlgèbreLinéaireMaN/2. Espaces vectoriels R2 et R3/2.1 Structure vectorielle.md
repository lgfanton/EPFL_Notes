# 2.1 Structure vectorielle

L'ensemble $\mathbb{R}^2$ (resp. $\mathbb{R}^3$) est muni d'une **multiplication scalaire** et d'une **addition** :
$$\mathbb{R}^2 = \{(x,y) \mid x, y \in \mathbb{R}\}$$

On peut y calculer des **combinaisons linéaires** (CL). Géométriquement (via le choix d'un repère), on peut visualiser ces opérations.

**Exemples** de vecteurs de $\mathbb{R}^2$ : $(0,0) = 0 v_1 = 0 v_2$,\ $(1,2)= v_1$,\ $(3,-1) = v_2$,\ $(2,4) = 2v_1$,\ $(1,-1/3)= \tfrac{1}{3}v_2$,\ $(4,1) = v_1 + v_2$,\ $(-1,5) = 2v_1 - v_2$, etc.
