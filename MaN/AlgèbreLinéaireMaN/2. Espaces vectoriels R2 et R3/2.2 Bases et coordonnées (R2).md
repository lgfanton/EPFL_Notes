# 2.2 Bases et coordonnées ($\mathbb{R}^2$)

Deux vecteurs $v_1, v_2 \in \mathbb{R}^2$ **non proportionnels** forment une **base** $\mathcal{B} = v_1, v_2$ de $\mathbb{R}^2$.

Tout $v \in \mathbb{R}^2$ s'écrit de façon **unique** $v = t_1 v_1 + t_2 v_2$. Les scalaires $t_1, t_2$ sont les **coordonnées** de $v$ dans $\mathcal{B}$ :
$$[v]_{\mathcal{B}} = \begin{pmatrix}t_1\\t_2\end{pmatrix}$$

**Exemple (a).** $v_1 = (1,0)$, $v_2 = (0,1)$ est la **base canonique** $\mathcal{B}_{can}$ de $\mathbb{R}^2$. Alors $v = (x,y) = x v_1 + y v_2$, donc $[v]_{\mathcal{B}_{can}} = \begin{pmatrix}x\\y\end{pmatrix}$ = les coordonnées canoniques.

**Exemple (b).** $v_1 = (0,2)$, $v_2 = (1/3, 0)$. La famille $\mathcal{B} = v_1, v_2$ est une base de $\mathbb{R}^2$ (vecteurs non proportionnels). On a $v = (x,y) = \frac{y}{2}(0,2) + 3x(\frac{1}{3},0)$, donc $[v]_{\mathcal{B}} = \begin{pmatrix}y/2\\3x\end{pmatrix}$.

> **⚠** $[v]_{\mathcal{B}} \neq v$ ! Les coordonnées dans $\mathcal{B}$ ne sont pas les composantes de $v$.
