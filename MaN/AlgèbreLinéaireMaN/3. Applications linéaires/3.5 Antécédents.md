# 3.5 Antécédents

**Définition.** Soit $w \in \mathbb{R}^p$. L'**ensemble des antécédents** de $w$ par $f$ est :
$$f^{-1}(\{w\}) = \{v \in \mathbb{R}^m \mid f(v) = w\} \subseteq \mathbb{R}^m$$

**Cas particulier :** $f^{-1}(\{0_{\mathbb{R}^p}\}) = \ker f$.

**Proposition.** On trouve l'ensemble des antécédents en résolvant un système. Pour $m=3$, $p=2$ :
$$v = (x,y,z) \in f^{-1}(\{(a,b)\}) \iff \begin{cases}\alpha_{1,1}x + \alpha_{1,2}y + \alpha_{1,3}z = a\\\alpha_{2,1}x + \alpha_{2,2}y + \alpha_{2,3}z = b\end{cases}$$

$$f^{-1}(\{w\}) = \begin{cases}\emptyset & \text{si } w \notin \text{Im}\, f \\ v_0 + \ker f & \text{si } w \in \text{Im}\, f\end{cases}$$

où $v_0$ est un antécédent particulier de $w$.
