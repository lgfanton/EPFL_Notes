# 3.3 Image

**Définition.** Soit $f : \mathbb{R}^m \to \mathbb{R}^p$ linéaire. L'**image** de $f$ est :
$$\text{Im}\, f = \{f(v) \mid v \in \mathbb{R}^m\} \subseteq \mathbb{R}^p$$

**Proposition.** $\text{Im}\, f$ est un **sous-espace vectoriel** de $\mathbb{R}^p$.

**Calcul.** $\text{Im}\, f = \text{Vect}(v_1, \ldots, v_r)$ où $v_1, \ldots, v_r$ sont les colonnes pivots de $A$ (les colonnes indépendantes). En particulier, $\text{rg}(f) = \dim(\text{Im}\, f) = \text{rg}(A)$.

**Exemple.** $f : \mathbb{R}^3 \to \mathbb{R}^2$, $(x,y,z) \mapsto (2x-3y+z,\; -4x+6y-2z)$.
$$A = \begin{pmatrix}2&-3&1\\-4&6&-2\end{pmatrix} = \begin{pmatrix}1\\-2\end{pmatrix}(2\ -3\ 1)$$
$\text{rg}(A) = 1$, donc $\text{Im}\, f = \text{Vect}((1,-2))$. En équation : $2x + y = 0$.
