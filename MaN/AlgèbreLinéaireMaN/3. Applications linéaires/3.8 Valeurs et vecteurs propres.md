# 3.8 Valeurs et vecteurs propres

**Définition.** Soit $f : \mathbb{R}^n \to \mathbb{R}^n$ linéaire. On dit que $\omega \in \mathbb{R}$ est une **valeur propre** de $f$ et $v \in \mathbb{R}^n \setminus \{\vec{0}\}$ un **vecteur propre** associé si :
$$f(v) = \omega v \qquad (\text{équiv.} : A_f v = \omega v)$$

L'**espace propre** associé à $\omega$ est :
$$R_\omega = \ker(f - \omega \cdot \text{id}) = \ker(A_f - \omega \mathbb{1}_n)$$

## Polynôme caractéristique

Les valeurs propres sont les racines du **polynôme caractéristique** :
$$\chi_f(\lambda) = \det(A_f - \lambda \mathbb{1}_n)$$

Pour $f : \mathbb{R}^2 \to \mathbb{R}^2$ avec $A_f = \begin{pmatrix}a&b\\c&d\end{pmatrix}$ :
$$\chi_f(\lambda) = \lambda^2 - \text{Tr}(A_f)\lambda + \det(A_f)$$

où $\text{Tr}(A_f) = a + d$.

## Exemples

**(a)** $f : \mathbb{R}^2 \to \mathbb{R}^2$, $(x,y) \mapsto (3x+3y,\ -x-y)$ (projection)

$A_f = \begin{pmatrix}3&3\\-1&-1\end{pmatrix}$, $\chi_f(\lambda) = \lambda^2 - 2\lambda = \lambda(\lambda - 2)$ → $\omega_1 = 0$, $\omega_2 = 2$

- $R_0 = \ker(A_f) = \text{vect}(3,-1)$ (directions annulées)
- $R_2 = \ker(A_f - 2\mathbb{1}) = \ker\begin{pmatrix}1&3\\-1&-3\end{pmatrix} = \text{vect}(3, -1)$...
