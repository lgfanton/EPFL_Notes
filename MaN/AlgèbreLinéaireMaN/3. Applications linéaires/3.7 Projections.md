# 3.7 Projections

**Définition.** Une application linéaire $f : \mathbb{R}^n \to \mathbb{R}^n$ est une **projection** ssi $f^2 = f$ (i.e. $A_f^2 = A_f$).

## Propriétés

- $\text{rg}(f) = r$ → on distingue les cas selon $r$ :
  - $r = 0$ : $f = 0$
  - $r = n$ : $f = \text{id}$, $A_f = \mathbb{1}_n$
  - $0 < r < n$ : projection proprement dite

**Remarque.** Si $f$ est une projection, $A_f$ est symétrique ssi $f$ projette **orthogonalement**.
Il existe aussi des projections non orthogonales.

## Résumé géométrique ($n=2$)

Si $f : \mathbb{R}^2 \to \mathbb{R}^2$ est une projection, $f$ projette $\mathbb{R}^2$ sur $\text{im}(f)$ **parallèlement à $\ker(f)$**.

| $r = 0$ | $r = 1$ | $r = 2$ |
|:---:|:---:|:---:|
| $f = 0$ | proj. sur une droite | $f = \text{id}$ |

## Proposition

$$\det(\lambda \cdot \text{id}) = \lambda^n \det(\text{id}) = \lambda^n$$

> **⚠** $\det(f+g) \neq \det(f) + \det(g)$ en général.

**Exemple.** $A_f = \begin{pmatrix}1&2\\3&6\end{pmatrix}$, $\det(f) = 6-6 = 0$, $\text{rg}(A_f) = 1$ : c'est une projection sur $\text{vect}\begin{pmatrix}1\\3\end{pmatrix}$.

Vérification : $A_f^2 = \begin{pmatrix}1&2\\3&6\end{pmatrix}^2 = \begin{pmatrix}7&14\\21&42\end{pmatrix} = 7 A_f$ — pas une projection car $A_f^2 \neq A_f$ !
