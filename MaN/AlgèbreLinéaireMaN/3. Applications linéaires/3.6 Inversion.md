# 3.6 Inversion

**Définition.** $f : \mathbb{R}^m \to \mathbb{R}^p$ est **inversible** si $\exists\, g : \mathbb{R}^p \to \mathbb{R}^m$ linéaire telle que $g \circ f = \text{id}_{\mathbb{R}^m}$ et $f \circ g = \text{id}_{\mathbb{R}^p}$. On note $g = f^{-1}$.

Cela nécessite $m = p$, $\text{rg}(f) = m$, et $\det A \neq 0$.

**Calcul.** Si $f$ est inversible, $A_{f^{-1}} = A_f^{-1}$.

**Exemple (a).** $f : \mathbb{R}^2 \to \mathbb{R}^2$, $(x,y) \mapsto (2x-y,\; -x+2y)$.
$$A = \begin{pmatrix}2&-1\\-1&2\end{pmatrix},\quad \det A = 3 \neq 0,\quad A^{-1} = \frac{1}{3}\begin{pmatrix}2&1\\1&2\end{pmatrix}$$
$$f^{-1}(a,b) = \tfrac{1}{3}(2a+b,\; a+2b)$$

**Exemple (b).** $f : \mathbb{R}^2 \to \mathbb{R}^2$, $(x,y) \mapsto (2x-y,\; x-\tfrac{1}{2}y)$.
$$A = \begin{pmatrix}2&-1\\1&-1/2\end{pmatrix},\quad \det A = 0 \Rightarrow f \text{ non inversible.}$$
