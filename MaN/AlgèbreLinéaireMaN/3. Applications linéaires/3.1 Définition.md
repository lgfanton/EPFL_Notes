# 3.1 Définition

On dit que $f : \mathbb{R}^m \to \mathbb{R}^p$ est **linéaire** si elle est associée à une matrice $A \in M_{p,m}(\mathbb{R})$ de la manière suivante. Par exemple avec $m = 3$, $p = 2$ :
$$f : \mathbb{R}^3 \to \mathbb{R}^2, \quad (x,y,z) \mapsto (\alpha_{1,1}x + \alpha_{1,2}y + \alpha_{1,3}z,\; \alpha_{2,1}x + \alpha_{2,2}y + \alpha_{2,3}z)$$
$$A = \begin{pmatrix}\alpha_{1,1}&\alpha_{1,2}&\alpha_{1,3}\\\alpha_{2,1}&\alpha_{2,2}&\alpha_{2,3}\end{pmatrix} \in M_{2,3}(\mathbb{R})$$

On a $[f(v)]_{\mathcal{B}_{can}} = A\,[v]_{\mathcal{B}_{can}}$. La matrice $A$ est appelée la **matrice de $f$ en base canonique**.

**Colonnes de $A$ = images des vecteurs de la base canonique :**
$$A = \Big([f(e_1)]_{\mathcal{B}_{can}} \;\Big|\; [f(e_2)]_{\mathcal{B}_{can}} \;\Big|\; \cdots \;\Big|\; [f(e_m)]_{\mathcal{B}_{can}}\Big)$$
