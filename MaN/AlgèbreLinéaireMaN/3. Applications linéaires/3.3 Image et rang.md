# 3.3 Image et rang

**Définition.** Soit $f : \mathbb{R}^n \to \mathbb{R}^p$ linéaire. L'**image** de $f$ est :
$$\text{im}(f) = \{f(v) \mid v \in \mathbb{R}^n\} \subseteq \mathbb{R}^p$$

**Proposition.** $\text{im}(f)$ est un sous-espace vectoriel de $\mathbb{R}^p$.

## Calcul de l'image

On a $\text{im}(f) = \text{vect}(v_1, \ldots, v_r)$ où $v_1, \ldots, v_r$ sont les colonnes pivots de $A_f$.

Le **rang** de $f$ est $\text{rg}(f) = \dim(\text{im}(f)) = \text{rg}(A_f)$.

## Exemples

**(a)** $f : \mathbb{R}^3 \to \mathbb{R}^2$, $(x,y,z) \mapsto (2x-3y+z,\ -4x+6y-2z)$

$\text{rg}(A_f) = 1$, $\text{im}(f) = \text{vect}((1,-2))$

En équation : $2x + y = 0$

**(b)** $f : \mathbb{R}^3 \to \mathbb{R}^3$, $(x,y,z) \mapsto (-2x+3y+z,\ x+y-3z,\ x-4y+2z)$

$$A_f = \begin{pmatrix}-2&3&1\\1&1&-3\\1&-4&2\end{pmatrix}$$

Décomposition colonne-ligne : $A_f = \begin{pmatrix}-2\\1\\1\end{pmatrix}(1,0,0) + \begin{pmatrix}3\\1\\-4\end{pmatrix}(0,1,0) + \begin{pmatrix}1\\-3\\2\end{pmatrix}(0,0,1)$

$\text{rg}(A_f)$ : échelonner pour trouver le rang et les colonnes pivots.

**(c)** $f : \mathbb{R}^2 \to \mathbb{R}^2$, $(x,y) \mapsto (x-y,\ -2x+2y)$

$\text{im}(f) = \text{vect}((1,-2))$ — en équation : $2x + y = 0$

$\ker(f) = \text{vect}((1,1))$ — en équation : $x - y = 0$

$A_f = \begin{pmatrix}1\\-2\end{pmatrix}(1,-1) \Rightarrow f(x,y) = (1,-2)(x-y)$
