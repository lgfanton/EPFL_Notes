# 3.7 Ensemble des antécédents

Ce paragraphe reprend §5 avec plus de détail. Soit $f : \mathbb{R}^m \to \mathbb{R}^p$ linéaire.

**Proposition.** Pour tout $w \in \mathbb{R}^p$ :
$$f^{-1}(\{w\}) = \begin{cases}\emptyset & \text{si } w \notin \text{Im}\, f \\ v_0 + \ker f & \text{si } w \in \text{Im}\, f\end{cases}$$

où $v_0$ est n'importe quel antécédent particulier ($f(v_0) = w$).

**Exemple.** $f : \mathbb{R}^3 \to \mathbb{R}^2$, $(x,y,z) \mapsto (2x-3y+z,\; -4x+6y-2z)$.

Cherchons $f^{-1}(\{(a,b)\})$. On a $A = \begin{pmatrix}1\\-2\end{pmatrix}(2\ -3\ 1)$.

- **Condition d'existence :** $(a,b) \in \text{Im}\, f = \text{Vect}((1,-2)) \iff \frac{a}{1} = \frac{b}{-2} \iff b = -2a$.
- Si $b = -2a$ : antécédent particulier $v_0 = (a/2, 0, 0)$.
- Solution générale : $f^{-1}(\{(a,b)\}) = (a/2,0,0) + \ker f$ (droite affine dans $\mathbb{R}^3$).
