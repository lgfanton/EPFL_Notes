# 3.1 Définitions

**Définition.** On dit que $f : \mathbb{R}^n \to \mathbb{R}^p$ est une **application linéaire** ssi $f$ peut s'écrire comme :
$$\mathbb{R}^n \ni (x_1,\ldots,x_n) \mapsto f(x_1,\ldots,x_n) = (a_1 x_1 + \cdots + a_n x_n,\ \ldots,\ a_p x_1 + \cdots)$$

où tous les coefficients $a_i \in \mathbb{R}$ — autrement dit, $f$ est une fonction dont chaque composante de sortie est une combinaison linéaire des entrées.

## Matrice associée

Chaque application linéaire $f : \mathbb{R}^n \to \mathbb{R}^p$ possède une **matrice associée** $A_f \in M(p \times n)$ :

$$A_f = \begin{pmatrix} a & b & c \\ d & e & f \end{pmatrix} \quad \text{pour } f : \mathbb{R}^3 \to \mathbb{R}^2$$

et $f(v) = A_f \cdot v$.

Les colonnes de $A_f$ sont les images des vecteurs de la base canonique :
$$A_f = \Big([f(e_1)]_{B.can} \;\Big|\; [f(e_2)]_{B.can} \;\Big|\; \cdots \;\Big|\; [f(e_n)]_{B.can}\Big)$$

**Exemple.** $f : \mathbb{R}^3 \to \mathbb{R}^2$, $(x,y,z) \mapsto (ax+by+cz,\ dx+ey+fz)$

$$A_f = \begin{pmatrix}a&b&c\\d&e&f\end{pmatrix}$$

## Matrice dans des bases quelconques

Si $B$ est une base de $\mathbb{R}^n$ et $B'$ une base de $\mathbb{R}^p$ :
$$[f]_{B,B'} = \Big([f(v_1)]_{B'} \;\Big|\; [f(v_2)]_{B'} \;\Big|\; \cdots \;\Big|\; [f(v_n)]_{B'}\Big)$$

et $[f(v)]_{B'} = [f]_{B,B'} \cdot [v]_B$.

**Relation avec la base canonique :**
$$[f]_{B,B'} = (P')^{-1} A_f P$$

où $P$ est la matrice de changement de base $B.can \to B$ et $P'$ celle de $B.can \to B'$.
