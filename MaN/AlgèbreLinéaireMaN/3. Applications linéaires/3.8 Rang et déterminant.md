# 3.8 Rang et déterminant

**Proposition.** Soit $f : \mathbb{R}^n \to \mathbb{R}^n$ linéaire de matrice $A$. Alors :
$$\det(\lambda f) = \det(\lambda A) = \lambda^n \det(A)$$

> **⚠** En général $\det(f + g) \neq \det(f) + \det(g)$.

**Contre-exemple :**
$$\begin{pmatrix}1&2\\3&5\end{pmatrix} = \begin{pmatrix}1&0\\3&0\end{pmatrix} + \begin{pmatrix}0&2\\0&5\end{pmatrix}$$
rg $= 2$, det $= -1$ ; rg $= 1$, det $= 0$ ; rg $= 1$, det $= 0$.

Le rang d'une somme n'est pas la somme des rangs. Les décompositions colonne-ligne montrent que :
$$\text{rg}(f+g) \le \text{rg}(f) + \text{rg}(g)$$
