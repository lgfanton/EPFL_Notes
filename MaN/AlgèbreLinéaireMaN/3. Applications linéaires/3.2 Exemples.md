# 3.2 Exemples

**(a)** $f : \mathbb{R}^2 \to \mathbb{R}^2$, $(x,y) \mapsto (2x - y,\; -x + 3y)$.
$$A = \begin{pmatrix}2&-1\\-1&3\end{pmatrix}$$

**(b)** $f : \mathbb{R}^3 \to \mathbb{R}^2$, $(x,y,z) \mapsto (x+y, 2x-z)$.
$$A = \begin{pmatrix}1&1&0\\2&0&-1\end{pmatrix}$$

**(c)** $f = \text{id}_{\mathbb{R}^m}$. La matrice est $I_m$.

**(d)** $f : \mathbb{R}^m \to \mathbb{R}^p$ l'application nulle. La matrice est $0$.
