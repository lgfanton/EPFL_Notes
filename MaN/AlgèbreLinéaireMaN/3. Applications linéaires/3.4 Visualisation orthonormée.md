# 3.4 Visualisation orthonormée

Dans ce paragraphe on visualise $\mathbb{R}^2$ par un repère du plan **orthonormé et orienté** $(O, \vec{e}_x, \vec{e}_y)$.

Pour $\theta \in \mathbb{R}$, on définit le vecteur unitaire :
$$v_\theta = (\cos\theta, \sin\theta), \qquad \|v_\theta\| = \sqrt{x^2+y^2} = 1$$

**Propriété de rotation.** $v_{\theta + \pi/2} = (-\sin\theta, \cos\theta)$ (rotation de $\frac{\pi}{2}$) :
$$\cos(\theta + \tfrac{\pi}{2}) = \cos(\tfrac{\pi}{2} - (-\theta)) = \sin(-\theta) = -\sin\theta$$
$$\sin(\theta + \tfrac{\pi}{2}) = \sin(\tfrac{\pi}{2} - (-\theta)) = \cos(-\theta) = \cos\theta$$

Et aussi :
- $v_{\theta - \pi/2} = (\sin\theta, -\cos\theta) = -v_{\theta+\pi/2}$
- $v_{\theta + \pi} = -v_\theta$
