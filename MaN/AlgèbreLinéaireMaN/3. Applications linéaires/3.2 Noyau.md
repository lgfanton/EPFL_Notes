# 3.2 Noyau

**Définition.** Soit $f : \mathbb{R}^n \to \mathbb{R}^p$ linéaire. Le **noyau** de $f$ est :
$$\ker(f) = \{v \in \mathbb{R}^n \mid f(v) = \vec{0}\}$$

C'est l'ensemble des vecteurs qui sont « écrasés » par $f$.

**Proposition.** $\ker(f)$ est un sous-espace vectoriel de $\mathbb{R}^n$.

## Calcul du noyau

$\ker(f) = \ker(A_f)$ = ensemble des solutions de $A_f \vec{x} = \vec{0}$.

On résout ce système homogène par échelonnement.

**Dimension du noyau.** On appelle $\dim(\ker(f))$ la **nullité** de $f$.

## Théorème du rang

$$\dim(\ker(f)) + \dim(\text{im}(f)) = n \qquad (\text{dimension du domaine})$$

## Exemples

**(a)** $f : \mathbb{R}^3 \to \mathbb{R}^2$, $(x,y,z) \mapsto (2x-3y+z,\ -4x+6y-2z)$

$$A_f = \begin{pmatrix}2&-3&1\\-4&6&-2\end{pmatrix} \xrightarrow{L_2 \leftarrow L_2+2L_1} \begin{pmatrix}2&-3&1\\0&0&0\end{pmatrix}$$

$\text{rg}(A_f) = 1$, donc $\ker(f) : 2x - 3y + z = 0 = \text{vect}\!\begin{pmatrix}1\\-2\\-1\end{pmatrix}\!\!,\!\begin{pmatrix}3\\2\\0\end{pmatrix}$ (ici $\ker$ est un plan).

**(b)** $f : \mathbb{R}^2 \to \mathbb{R}^2$, $(x,y) \mapsto (x-y,\ -2x+2y)$

$$A_f = \begin{pmatrix}1&-1\\-2&2\end{pmatrix},\quad \ker(f) : x - y = 0 = \text{vect}((1,1))$$
