# 6.6 Forme réduite (cas $\Delta < 0$)

Quand $\Delta < 0$, $\chi_f$ n'a pas de racines réelles : $f$ n'a **pas de valeurs propres réelles** et n'est pas diagonalisable sur $\mathbb{R}$.

Les "valeurs propres complexes" s'écrivent $\omega \pm i\xi$ avec $\omega = \frac{\text{Tr}(A)}{2}$ et $\xi = \frac{\sqrt{-\Delta}}{2} > 0$.

**Forme réduite.** Il existe une base $\mathcal{B}$ telle que :
$$[f]_\mathcal{B} = \rho R_\theta = \begin{pmatrix}\omega&-\xi\\\xi&\omega\end{pmatrix}$$

où $\rho = \sqrt{\omega^2 + \xi^2} = \sqrt{\det A}$ et $\theta = \text{sgn}(\xi)\arccos(\omega/\rho)$.

Géométriquement : $f$ est la composée d'une **rotation** d'angle $\theta$ et d'une **homothétie** de rapport $\rho$.

**Procédé :**
1. Calculer $\omega = \frac{\text{Tr}(A)}{2}$, $\xi = \frac{\sqrt{-\Delta}}{2}$.
2. Prendre $v_1$ quelconque $\neq 0$.
3. Poser $v_2 = \frac{1}{\xi}(f - \omega\, \text{id})(v_1)$.
4. $\mathcal{B} = v_1, v_2$ donne $[f]_\mathcal{B} = \begin{pmatrix}\omega&-\xi\\\xi&\omega\end{pmatrix}$.

**Application : $A^n$.**

$$A^n = P\begin{pmatrix}\omega&-\xi\\\xi&\omega\end{pmatrix}^n P^{-1} = P\,\rho^n\begin{pmatrix}\cos(n\theta)&-\sin(n\theta)\\\sin(n\theta)&\cos(n\theta)\end{pmatrix}P^{-1}$$

**Exemple.** $f : \mathbb{R}^2 \to \mathbb{R}^2$, $(x,y) \mapsto (x+2y, -2x+y)$.
$$A = \begin{pmatrix}1&2\\-2&1\end{pmatrix},\quad \text{Tr} = 2,\quad \det = 5,\quad \Delta = 4 - 20 = -16 < 0$$
$\omega = 1$, $\xi = 2$, $\rho = \sqrt{5}$, $\theta = \arctan(2/1) = \arctan 2$.

Prendre $v_1 = (1,0)$, $v_2 = \frac{1}{2}((1,0+2\cdot0) - (1,0)) = (0,-1)$...
