# 6.3 Vecteur propre et base propre

**Définition.** Un **vecteur propre** de $f$ associé à la valeur propre $\omega$ est un vecteur $v \neq 0$ tel que :
$$f(v) = \omega v \qquad (\iff (A - \omega I_2)v = 0 \iff v \in \ker(f - \omega\, \text{id}_{\mathbb{R}^2}))$$

L'**espace propre** associé à $\omega$ est $R_\omega = \ker(A - \omega I_2)$.

**Proposition.** $\omega$ est valeur propre de $f$ $\iff$ $\det(A - \omega I_2) = 0$ $\iff$ $R_\omega \neq \{0\}$.

**Une base $\mathcal{B}$ est dite propre pour $f$** si elle est constituée de vecteurs propres.

**Exemple (a).** $A = \frac{1}{4}\begin{pmatrix}5&9\\3&-1\end{pmatrix}$, valeurs propres $\omega_1 = -1$, $\omega_2 = 2$.

- $R_{-1} = \ker(A + I_2) = \ker\frac{1}{4}\begin{pmatrix}9&9\\3&3\end{pmatrix} = \text{Vect}((1,-1))$
- $R_2 = \ker(A - 2I_2) = \ker\frac{1}{4}\begin{pmatrix}-3&9\\3&-9\end{pmatrix} = \text{Vect}((3,1))$

Base propre $\mathcal{B} = (1,-1),(3,1)$ :
$$[f]_\mathcal{B} = \begin{pmatrix}-1&0\\0&2\end{pmatrix} = R \qquad P = \begin{pmatrix}1&3\\-1&1\end{pmatrix}$$

**Vérification :** $f(1,-1) = (-1)(1,-1)$ ✓ et $f(3,1) = 2(3,1)$ ✓.
