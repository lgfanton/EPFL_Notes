# 6.5 Forme réduite (cas $\Delta = 0$)

Quand $\Delta = 0$, $\chi_f$ a une racine double $\omega$ et $\text{Tr}(A) = 2\omega$.

## Sous-cas : $f = \omega\, \text{id}_{\mathbb{R}^2}$

$A = \omega I_2$. Toute base est propre : $[f]_\mathcal{B} = \omega I_2$ dans n'importe quelle base.

## Sous-cas général : $f \neq \omega\, \text{id}_{\mathbb{R}^2}$

$\dim R_\omega = 1$ (un seul vecteur propre).

**Procédé ("machine à laver") :**
1. Prendre $v_2$ un vecteur **non propre** pour $f$ (n'importe quel $v_2 \notin R_\omega$, $v_2 \neq 0$).
2. Poser $v_1 = (f - \omega\, \text{id})(v_2) = Av_2 - \omega v_2 \neq 0$ (c'est un vecteur propre).
3. $\mathcal{B} = v_1, v_2$ et $\mathcal{B}' = v_2, v_1$ donnent :

$$[f]_\mathcal{B} = \begin{pmatrix}\omega&0\\1&\omega\end{pmatrix}, \qquad [f]_{\mathcal{B}'} = \begin{pmatrix}\omega&1\\0&\omega\end{pmatrix}$$

La forme $\begin{pmatrix}\omega&1\\0&\omega\end{pmatrix}$ est appelée **bloc de Jordan** (forme réduite non diagonale).

**Exemple.** $f : \mathbb{R}^2 \to \mathbb{R}^2$, $(x,y) \mapsto (5x+3y, -3x-y)$.
$$A = \begin{pmatrix}5&3\\-3&-1\end{pmatrix},\quad \chi_f(X) = (X-2)^2,\quad \omega = 2,\quad \Delta = 0$$
- Prendre $v_2 = (1,0)$ (non propre car $f(1,0) = (5,-3) \neq 2(1,0)$)
- $v_1 = (A-2I_2)v_2 = \begin{pmatrix}3\\-3\end{pmatrix}$, donc $v_1 = (1,-1)$ (après normalisation)
- $[f]_{\mathcal{B}'} = \begin{pmatrix}2&1\\0&2\end{pmatrix}$ avec $\mathcal{B}' = (1,-1),(1,0)$
