# 1.2 Suites majorées, minorées, monotones

## Bornitude

**Définitions 1.3.** Soit $(a_n)$ une suite.

- $(a_n)$ est **majorée** s'il existe $M \in \mathbb{R}$ tel que $a_n \leq M$ pour tout $n$. $M$ est un **majorant**.
- $(a_n)$ est **minorée** s'il existe $m \in \mathbb{R}$ tel que $a_n \geq m$ pour tout $n$. $m$ est un **minorant**.
- $(a_n)$ est **bornée** si elle est majorée et minorée, i.e. $\exists C > 0,\ |a_n| \leq C$ pour tout $n$.

**Exemples 1.4.**
- $a_n = \frac{1}{n}$ est bornée : $0 < a_n \leq 1$.
- $a_n = n^2$ est minorée (par 1) mais pas majorée.
- $a_n = (-1)^n \cdot \frac{n-1}{n}$ est bornée : $|a_n| < 1$.
- $a_n = \frac{6n+100}{n+10}$ est minorée : $a_n \geq 6$.
- $a_n = \sqrt{n}$ est minorée (par 0) mais pas majorée.

**Remarque 1.5.** Pour majorer un quotient $\frac{A}{B}$ avec $A, B > 0$, chercher $\tilde{A} \geq A$ et $0 < \tilde{B} \leq B$ puis écrire $\frac{A}{B} \leq \frac{\tilde{A}}{\tilde{B}}$.

### 1.2.1 Monotonicité

**Définitions 1.6.**
- $(a_n)$ est **croissante** si $a_n \leq a_{n+1}$ pour tout $n$.
- $(a_n)$ est **strictement croissante** si $a_n < a_{n+1}$ pour tout $n$.
- $(a_n)$ est **décroissante** si $a_n \geq a_{n+1}$ pour tout $n$.
- $(a_n)$ est **strictement décroissante** si $a_n > a_{n+1}$ pour tout $n$.
- $(a_n)$ est **monotone** si elle est croissante ou décroissante.

**Méthodes pour étudier la monotonie :**
1. Signe de $a_{n+1} - a_n$
2. Rapport $\frac{a_{n+1}}{a_n}$ (si tous les termes sont positifs) : $> 1$ ⟹ croissante, $< 1$ ⟹ décroissante
3. Étude de la fonction $f$ telle que $a_n = f(n)$

**Exemple.** $a_n = \frac{3n+4}{n+1}$ est décroissante :
$$a_n - a_{n+1} = \frac{3n+4}{n+1} - \frac{3(n+1)+4}{(n+1)+1} = \frac{1}{(n+1)(n+2)} > 0$$
