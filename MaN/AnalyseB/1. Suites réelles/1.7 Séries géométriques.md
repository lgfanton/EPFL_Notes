# 1.7 Séries géométriques

## Somme partielle

La **somme géométrique** de raison $q \neq 1$ est :
$$\sum_{k=0}^{n} q^k = 1 + q + q^2 + \cdots + q^n = \frac{1 - q^{n+1}}{1 - q}$$

**Preuve.** Poser $S = 1 + q + \cdots + q^n$, puis $qS = q + q^2 + \cdots + q^{n+1}$. Donc $S - qS = 1 - q^{n+1}$.

## Série géométrique infinie

Si $|q| < 1$ :
$$\sum_{k=0}^{+\infty} q^k = \frac{1}{1-q}$$

**Exemple.** $\displaystyle\sum_{k=0}^{\infty} \left(\frac{1}{2}\right)^k = \frac{1}{1 - \frac{1}{2}} = 2$.

Si $|q| \geq 1$ (et $q \neq 1$), la série diverge.

## Variantes

$$\sum_{k=0}^{n} q^k = \frac{q^{n+1}-1}{q-1}, \qquad \sum_{k=m}^{n} q^k = q^m \cdot \frac{1-q^{n-m+1}}{1-q}$$
