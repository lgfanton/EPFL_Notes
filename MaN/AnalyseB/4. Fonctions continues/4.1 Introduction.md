# 4.1 Introduction

**Définition.** $f$ est **continue en $x_0$** si :
$$\lim_{x \to x_0} f(x) = f(x_0)$$

Ceci requiert :
1. $f$ définie en $x_0$
2. $\lim_{x \to x_0} f(x)$ existe
3. La limite est égale à $f(x_0)$

$f$ est **continue sur $D$** si elle est continue en tout point de $D$.

## Propriétés

- Toute **fonction polynomiale** est continue sur $\mathbb{R}$.
- Toute **fraction rationnelle** est continue sur son domaine de définition.
- $\sin$, $\cos$, $\exp$, $\ln$ sont continues sur leur domaine.
- **Somme, produit, quotient** (dénominateur $\neq 0$) de fonctions continues est continue.
- **Composée** de fonctions continues est continue.

## Discontinuités

| Type | Description |
|------|-------------|
| Discontinuité removable | $\lim_{x\to x_0} f(x)$ existe mais $\neq f(x_0)$ |
| Saut | $\lim^+ \neq \lim^-$ |
| Infinie | $\lim_{x\to x_0} f(x) = \pm\infty$ |
| Essentielle | Pas de limite |
