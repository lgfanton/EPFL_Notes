# 2.1 Introduction

Une **fonction réelle** est une application $f : D \to \mathbb{R}$ avec $D \subseteq \mathbb{R}$.

- **Domaine de définition** $D_f$ : ensemble des $x$ pour lesquels $f(x)$ est défini.
- **Image** (ou codomaine) : $f(D) = \{f(x) \mid x \in D\}$.
- **Graphe** : $\mathcal{G}_f = \{(x, f(x)) \mid x \in D_f\} \subseteq \mathbb{R}^2$.

**Opérations :**
- Somme : $(f+g)(x) = f(x) + g(x)$, sur $D_f \cap D_g$
- Produit : $(fg)(x) = f(x)g(x)$, sur $D_f \cap D_g$
- Quotient : $\left(\frac{f}{g}\right)(x) = \frac{f(x)}{g(x)}$, sur $D_f \cap D_g \cap \{g \neq 0\}$
