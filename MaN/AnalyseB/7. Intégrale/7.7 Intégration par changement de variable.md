# 7.7 Intégration par changement de variable

**Théorème.** Si $\varphi : [\alpha, \beta] \to [a,b]$ est dérivable et $f$ continue sur $[a,b]$ :
$$\int_a^b f(x)\,dx = \int_\alpha^\beta f(\varphi(t))\,\varphi'(t)\,dt$$

avec $a = \varphi(\alpha)$ et $b = \varphi(\beta)$.

**En pratique (substitution $x = \varphi(t)$) :**
$$dx = \varphi'(t)\,dt, \qquad \text{et changer les bornes}$$

**Exemple.**
$$\int_0^1 \frac{x}{\sqrt{1+x^2}}\,dx$$

Poser $u = 1 + x^2$, $du = 2x\,dx$. Bornes : $x=0 \Rightarrow u=1$, $x=1 \Rightarrow u=2$.
$$= \int_1^2 \frac{1}{\sqrt{u}} \cdot \frac{du}{2} = \frac{1}{2}\Big[2\sqrt{u}\Big]_1^2 = \sqrt{2} - 1$$
