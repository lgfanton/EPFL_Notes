# 7.1 Construction de l'intégrale Riemann–Darboux

## Sommes de Darboux

Soit $f : [a, b] \to \mathbb{R}$ bornée. Pour une subdivision $a = x_0 < x_1 < \cdots < x_n = b$ :

$$m_k = \inf_{x \in [x_{k-1}, x_k]} f(x), \qquad M_k = \sup_{x \in [x_{k-1}, x_k]} f(x)$$

**Somme inférieure :** $\underline{S}(f) = \sum_{k=1}^n m_k (x_k - x_{k-1})$

**Somme supérieure :** $\overline{S}(f) = \sum_{k=1}^n M_k (x_k - x_{k-1})$

On a toujours $\underline{S}(f) \leq \overline{S}(f)$.

## Intégrabilité au sens de Riemann–Darboux

$f$ est **intégrable sur $[a,b]$** si :
$$\sup_{\text{subdivisions}} \underline{S}(f) = \inf_{\text{subdivisions}} \overline{S}(f)$$

La valeur commune est l'**intégrale** de $f$ sur $[a,b]$, notée $\displaystyle\int_a^b f(x)\,dx$.

**Théorème.** Toute fonction continue sur $[a,b]$ est intégrable.

**Interprétation géométrique.** Pour $f \geq 0$, $\displaystyle\int_a^b f(x)\,dx$ est l'aire sous la courbe $y = f(x)$ entre $x = a$ et $x = b$.
