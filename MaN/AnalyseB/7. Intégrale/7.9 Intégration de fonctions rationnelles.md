# 7.9 Intégration de fonctions rationnelles

Une **fraction rationnelle** est $\frac{P(x)}{Q(x)}$ avec $P, Q$ polynômes.

### 7.9.1 Méthode générale

1. **Si $\deg P \geq \deg Q$** : effectuer la division euclidienne :
$$\frac{P(x)}{Q(x)} = S(x) + \frac{R(x)}{Q(x)}, \quad \deg R < \deg Q$$

2. **Factoriser $Q(x)$** en facteurs irréductibles sur $\mathbb{R}$ :
$$Q(x) = a \prod_i (x - r_i)^{m_i} \prod_j (x^2 + p_j x + q_j)^{n_j}$$

3. **Décomposer en éléments simples** :
   - Pour chaque facteur $(x-r)^m$ : termes $\dfrac{A_1}{x-r} + \dfrac{A_2}{(x-r)^2} + \cdots + \dfrac{A_m}{(x-r)^m}$
   - Pour chaque facteur irréductible $(x^2+px+q)^n$ : termes $\dfrac{B_k x + C_k}{(x^2+px+q)^k}$

4. **Intégrer** chaque terme simple.

### 7.9.2 Recherche des coefficients

- Méthode des valeurs particulières (substituer des valeurs de $x$).
- Identification des coefficients de chaque puissance de $x$.

**Exemples de primitives simples :**

$$\int \frac{dx}{x-a} = \ln|x-a| + C$$
$$\int \frac{dx}{(x-a)^n} = \frac{-1}{(n-1)(x-a)^{n-1}} + C \quad (n \geq 2)$$
$$\int \frac{x\,dx}{x^2+a^2} = \frac{1}{2}\ln(x^2+a^2) + C$$
$$\int \frac{dx}{x^2+a^2} = \frac{1}{a}\arctan\!\left(\frac{x}{a}\right) + C$$
