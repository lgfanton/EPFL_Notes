# 7.8 Changement de variable : fonctions trigonométriques

## Substitution $x = \sin\theta$ (ou $\cos\theta$)

Utile pour les intégrales contenant $\sqrt{1-x^2}$.

$$x = \sin\theta,\quad dx = \cos\theta\,d\theta,\quad \sqrt{1-x^2} = \cos\theta$$

## Substitution $x = \tan\theta$

Utile pour les intégrales contenant $\frac{1}{1+x^2}$ ou $\sqrt{1+x^2}$.

$$x = \tan\theta,\quad dx = \frac{d\theta}{\cos^2\theta},\quad 1+x^2 = \frac{1}{\cos^2\theta}$$

## Substitution de Weierstrass : $t = \tan\!\left(\frac{x}{2}\right)$

Transforme toute expression rationnelle en $\sin x$, $\cos x$ en une fraction rationnelle en $t$ :

$$\sin x = \frac{2t}{1+t^2}, \qquad \cos x = \frac{1-t^2}{1+t^2}, \qquad dx = \frac{2\,dt}{1+t^2}$$

**Exemple.**
$$\int \frac{dx}{1 + \cos x} \xrightarrow{t = \tan(x/2)} \int \frac{1}{1 + \frac{1-t^2}{1+t^2}} \cdot \frac{2\,dt}{1+t^2} = \int \frac{1}{1+t^2} \cdot (1+t^2)\,dt = \int dt = t + C = \tan\!\left(\frac{x}{2}\right) + C$$
