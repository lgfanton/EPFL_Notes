# 7.12 Longueurs d'arcs

### 7.12.1 Longueur du graphe d'une fonction

La longueur du graphe de $f$ sur $[a,b]$ est :
$$L = \int_a^b \sqrt{1 + [f'(x)]^2}\,dx$$

**Exemple.** Longueur du graphe de $f(x) = x^{3/2}$ sur $[0,1]$ :
$$L = \int_0^1 \sqrt{1 + \frac{9x}{4}}\,dx$$

### 7.12.2 Longueur d'une courbe paramétrée

La longueur d'un arc paramétré $M(t) = (x(t), y(t))$ pour $t \in [\alpha, \beta]$ est :
$$L = \int_\alpha^\beta |\dot{\vec{r}}(t)|\,dt = \int_\alpha^\beta \sqrt{[\dot{x}(t)]^2 + [\dot{y}(t)]^2}\,dt$$

**Vitesse instantanée.** On appelle $v(t) = |\dot{\vec{r}}(t)| = \sqrt{\dot{x}^2 + \dot{y}^2}$ la **vitesse scalaire** de la particule.
