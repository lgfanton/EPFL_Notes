# 7.4 Primitives

$F$ est une **primitive** de $f$ sur $I$ si $F'(x) = f(x)$ pour tout $x \in I$.

Si $F$ est une primitive de $f$, alors toutes les primitives sont de la forme $F(x) + C$ ($C \in \mathbb{R}$).

## Table des primitives usuelles

| $f(x)$ | $F(x)$ |
|--------|--------|
| $x^n$ ($n \neq -1$) | $\dfrac{x^{n+1}}{n+1}$ |
| $\dfrac{1}{x}$ | $\ln|x|$ |
| $e^x$ | $e^x$ |
| $a^x$ | $\dfrac{a^x}{\ln a}$ |
| $\sin x$ | $-\cos x$ |
| $\cos x$ | $\sin x$ |
| $\dfrac{1}{\cos^2 x}$ | $\tan x$ |
| $\dfrac{1}{\sqrt{1-x^2}}$ | $\arcsin x$ |
| $\dfrac{1}{1+x^2}$ | $\arctan x$ |
| $\dfrac{1}{\sqrt{x}}$ | $2\sqrt{x}$ |

### 7.4.1 Recherche des primitives

Les primitives ne s'expriment pas toujours en termes de fonctions élémentaires (ex. : $e^{-x^2}$, $\frac{\sin x}{x}$).
