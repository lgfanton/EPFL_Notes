# 7.13 Surfaces de révolution

La **surface de révolution** obtenue en faisant tourner le graphe de $f$ autour de l'axe $Ox$ a pour aire :
$$S = 2\pi \int_a^b |f(x)| \sqrt{1 + [f'(x)]^2}\,dx$$

**Pour une courbe paramétrée** $M(t) = (x(t), y(t))$ tournant autour de $Ox$ :
$$S = 2\pi \int_\alpha^\beta |y(t)| \sqrt{[\dot{x}(t)]^2 + [\dot{y}(t)]^2}\,dt$$

**Exemple.** Surface d'une sphère de rayon $R$ : faire tourner $f(x) = \sqrt{R^2-x^2}$ sur $[-R,R]$ :
$$S = 2\pi \int_{-R}^R \sqrt{R^2-x^2} \cdot \frac{R}{\sqrt{R^2-x^2}}\,dx = 2\pi R \int_{-R}^R dx = 4\pi R^2$$
